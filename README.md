# pong
Used JSWING to create a pong application. There may be a few bugs here and there.

Issue:
There's a minor bug where the top part of player 1 & player 2 allows the ball to pass through.

Controls
  Player 1:
    - use W & S to move up & down.
  Player 2:
    - use up & down arrow keys to move up and down.
